# UI and API Automation Framework

Automation framework built using:

- Java
- Selenium WebDriver
- Cucumber BDD
- RestAssured
- TestNG
- Page Object Model
- PageFactory

## Features

UI Automation
- Page Object Model with PageFactory
- Modular Page Classes
- Utility classes for waits and parsing

API Automation
- RestAssured based API tests
- Service layer implementation
- Response validations

## Framework Structure

mission
- pages
- utils
- api
- step definitions

## Execution

Run tests using Maven:

mvn clean test
